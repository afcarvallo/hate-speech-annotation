Welcome to the Hate Speech Annotation Guide

Introduction

Hate speech is any form of expression that incites hatred or violence against a group based on characteristics like race, religion, gender, or sexual orientation. It can be explicit (direct insults or threats) or implicit (coded language or stereotypes).

Annotation Process Overview

Pre-annotation: Data is pre-processed using a Language Model (LLM) to highlight potential hate speech.
Consistency: Your role will consist of reviewing annotations made by the LLM.

General Rules

Positive, negative and multi-word rules that apply to all mention labels, including general spelling rules.

  • Do not include spaces, special characters or punctuation marks that appear before or after each mention in the label.
  • Sometimes, the to-be-annotated text fragment might go beyond the boundaries of a single sentence.
  • One should bear that in the same sentence more than one annotation can be used.

    • Positive Rules

    Rules specifying the entities that should be annotated:

  • Every mention to groups in the following categories: ethnicity, religion, sexual orientation, gender, disability, intersectional. These labels will start with the prefix G.
  • Every offensive mention to groups in the preceding categories. These labels will start with the prefix I.
  • Every offensive/insulting word.
  • Every attack intent in the following categories: dehumanization, derogation, animosity, threatening, pro-hate. Generally speaking, attack intents correspond to text fragments (spans).
  • There can be more than one entity in one sentence
  • There can be annotations inside annotations. For example a group mentioned inside an intent.
  • Words containing orthographic errors or typos should be annotated as well.

    • Negative Rules

    Rules specifying the entities that should be annotated:

  • Every mention to groups not considered in the preceding categories.
  • Negative connotative adjective, e.g., you are so lazy
  • Words that are implicit attacks in a context, but not out of context. e.g. “parasite” out of context is not an insult. E.g. These dishes are full of parasites.

    • Multi-word Rules

    Rules specifying whether a group of words should be annotated under a single label.

  • Both groups, attacks and offensive mentions can be composed by multi-words (n-grams).
